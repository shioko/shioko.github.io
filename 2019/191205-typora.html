围绕苏菲狗的废物利用行动

发表于 2019-12-05 | 更新于 2019-12-06 | 分类于 汐喵晒鱼
本文字数： 3.2k | 阅读时长 ≈ 7 分钟 | 评论数：

　　最近的码字环境：工作站常年开机，只要挂在登录界面就可以把文件共享到苏菲狗上，但是工作站看不到苏菲狗共享的文件（可能是 win10 不能向下兼容 win7），而且千反田的电脑上也看不见（可能因为他的系统是盗版），但总之他看不见也是一件好事，我怕这个连声卡驱动都不会装的电脑小白看见了我的共享文档之后大喊一声“你在我电脑上放了什么东西鸭”然后直接潇洒地扔进回收站，那同居生活应该就宣告终结了【
　　因为实在忍受不了 sublime text 2 在 win10 + 高分屏下的狗啃字体渲染下载了 typora，安装完一看，哦豁，200+M，果然爱美是要付出代价的……
　　不过好看也是真的好看，开着侧边栏可以查看文件列表和大纲就很有当年用有道云笔记码字的感觉，不过还是有些不太适应使用习惯的部分 :-cmx-12-: 然后顺便写一下围绕苏菲狗做的废物利用行动……


.+†+.

　　苏菲狗入手已经快一整年了，除了不知道是我动了什么设置导致翻开键盘盖无法唤醒屏幕以外没有出现什么太大的问题。但是今年一整年我几乎都在质问自己，像我这种搬家一年半出门总时长小于 40 小时且基本杜绝了实体社交的居家死宅总裁漫民工到底买它干嘛？一直到冬天来了才终于再次发现了它的价值：工作站太重了，我需要一台能架在折叠桌上窝在被窝里码字的 windows 电脑。
　　但其实 windows 这个需求也是被创造创造出来的。我以前用安卓平板 + 蓝牙键盘码字一样很舒服，橘子写作还可以自动首行加两个全角空格，还有字数统计和码字速度统计，除开云同步服务器挂了以外实在是个很好用的 app。不过安卓传文件到电脑上确实有点复杂，另外现在所有的文稿都是要传到 hexo 上的， markdown 也很难安卓上排版，windows 又还是更加方便一点。
　　所以“用 windows 码字”这个需求其实和苏菲狗本身一样，弃之可惜食之无味。但既然买都买了，挂闲鱼又真的很麻烦，我前年在闲鱼上挂的珠宝盒 lo 鞋和 M 码三叶亭裙子至今没有成功卖出去，几十块钱的东西还要我给你顺丰包邮，我干嘛不白送你算了？

　　以下是苏菲狗的弱智行为以及我尝试补救的措施：

  • 触控笔的手感像拧松了螺纹的中性笔，完全没办法写字
    • 买了 SZYMT 家的笔尖和类纸膜，我也不知道我拼对了没有反正他们家的名字好难打可以正常在屏幕上画画了，但是 SAI 缺少触屏优化，实际画起来还是很困难，我得把屏幕竖起来再一边按键盘……反正画了两天我的腱鞘炎又光荣复发 :-cmx-11-:
  • UWP 网易云不能隐藏窗口，而且总是放着放着就哑巴了
    • 无解【sb 网易
  • sublime text 的字体渲染效果过于感人
    • 换了 typora
  • windows 应用基本都没做触屏优化，至少我常用的那些没有……
    • 装了个 Gesture Sign，正在研究中……不过我不画画的时候一般还是用键盘比较多

　　最后，这条可能是我的弱智行为……我一直以为苏菲狗没有键盘背光，直到一个月前芒果在我的 F7 键上踩了一下 emmmmm

typora 使用体验

　　第一印象是界面好看！下载前兴致勃勃地试图用 css 自制一个主题，在打开本地主题的一瞬间放弃了……wdnmd CSS 是真的又臭又长啊【
　　另外就是有些不太符合我操作习惯的地方……

  • 文字下方留白太少，sublime text 是可以无限下拉的，我平时至少要留 1/3 左右的空白，因为低着头打字实在太累了……
  • 回车自动空一行，虽然我知道这是 markdpwn 的标准语法，但我码字的时候“换行”跟“空一行”是完全不同的两个操作，虽然按 Shift+回车 可以实现单回车换行，可是这个快捷键又跟输入法冲突了，总之很难受【
  • 针对触控的优化还是很少，绝大多数功能都需要通过下拉菜单或着快捷键实现。所以我试了一下无边窗口模式还是放弃了……绝大多数功能都没有做 UI 按钮，没有菜单栏我实在是寸步难行【
  • 还有新建 hexo 文档打开之后第一行输入两个全角空格会自动跳到分割线上面去，我不知道这是 bug 还是什么……总之第一行必须在代码模式下才能输入成功，然后代码模式里又不能打开侧边栏，虽然我知道你们程序员是不会手动输这两个空格甚至根本连首行缩进都不会做的，可是我很难受啊！！！(╯°Д°)╯︵┴┴
  • 侧边栏很美好，但请问为什么每次最小化以后它就消失了……【于是我为这个侧边栏特意设置了一个屏幕手势，我简直是真爱【

　　然后是优点：

  • 好看，真的好看，字体渲染非常平滑，我甚至感觉不到我在用 sb win10【
  • 侧边栏可以显示文件列表和大纲，虽然大纲只是单纯地提取了 markdown 的标题元素不能手动加备注，不过还是很实用的
  • 有中文！而且支持中文的字数统计，我终于不用为了字数统计花 30s 打开 word 了……【话虽如此试了一下在苏菲狗上打开 typora 也需要 5 秒钟，sublime text 需要 10 秒……算了算了，废物利用，废物利用 :-cmx-11-:
  • 实时渲染很平滑，我可以告别一边码字一边 hexo s 的日常了

　　应该还有很多功能我暂时没有发现的？唯一的缺点就是太大了……我没搞懂为什么 mac 版安装完毕也才 20+M，windows 版光一个安装包就 50+M了，就歧视田厂呗？？

类纸膜与笔尖

　　上个月购入了 SZYMT 家的类纸膜和笔尖，收到的第一天就感激涕零地手写了两屏的评价给店家贴上。不过你们家的名字也太难记了吧，我只能记成 苏子阳美瞳 ……【
　　然而用来画画依旧效果感人。首先 SAI 没有触控优化，高分屏下的显示效果又非常感人，而且笔触还有无法忽视的延迟……日常摸摸鱼还可以，拿来勾线出成品还是要放到大电脑上去，或者至少插个我靠。
　　真的我觉得苏菲狗这块屏幕颜色还挺好看的……可惜其它硬件太差了，如果给我一个后悔的机会，我还是会买 ipad【

　　顺带一提昨天芒果把我的笔尖叼出来扒拉玩不见了，我一度以为她吞下去了，拿吸尘器和铷磁铁把整个房间吸了一遍无果。我绝望地思考要不要把手上的代肝单子转掉两天带她去医院以及这次开刀会不会又凝血障碍住半个月过年医院要不要收我 3 倍住院费我存着买 ipad 的 6k 块钱够不够花等等一系列问题长达两个小时，然后喂她吃完晚饭放她出来玩了一会，突然发现地上有一条又细又短的黑线，我走过去一看，果然是我的笔芯！！！
　　揪着她骂了一顿顺便掏耳朵泄愤，老实了大概 18 个小时，现在又开始在我的键盘上散步顺便扒拉扒拉我的充电线。这猫太臭了我不想养了！！！

Gesture Sign 调教笔记

　　简单实用的触屏手势小软件，基本上没什么好说的？
　　不过中间确实遇到一个问题，就是苏菲狗键盘最上面一排的编辑键都被绑定了快捷功能，手势动作里没有 关闭窗口 这个选项，我只能通过 输入键盘快捷键 alt + F4 实现，可是按 F4 只能静音不能触发快捷键，这就很头疼……
　　查了一下要把 Fn 键的灯按亮了才能释放功能键，然后我自己试了一下，连按 Fn + Alt + F4 也是可以关闭窗口的，而且操作完毕之后 Fn 键的灯也会保持关闭，再按F4 还是正常的静音功能。
　　然而录制快捷键的时候 Fn 并不会显示在记录框里……一路折腾到系统自带的手势设置，发现和 GS 一样，没有关闭窗口动作只能发送快捷键，然后 Fn 键不会被录入，但是我自己试了一下，虽然 Fn 键没有出现在录入框里，但是它实际是被录入进去了的……
　　于是倒回到 GS，录入快捷键 Fn + Alt + F4，虽然只显示了 alt + F4，但实际操作成功了。不知道是不是系统的 bug……果然 windows 真的好难用啊【
