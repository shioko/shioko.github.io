割裂

发表于 2020-08-24 | 更新于 2020-10-01 | 分类于 汐喵晒鱼
本文字数： 1.5k | 阅读时长 ≈ 3 分钟 | 评论数：

　　好消息是这周我终于画出了能发的东西。
　　以及距上次更新似乎只过了一周。另外庆祝我的 blog 不知道什么时候突破了 1w 访客，虽然应该都是冲着之前那些代码笔记来的，结果我今年都在发一些毫无意义的怨妇文学，也是让大家见笑了【扶额
　　坏消息我越来越不想说话了。


.+†+.

　　之前看见过一个说法是语言不是一种知识而是一种器官的能力，心脏会跳动肌肉会伸缩，语言器官也会说话，而这一切都是需要在长期保持运动的状态下才能维持的，而我偏偏找不到人说话。
　　没有倾诉的对象也没有记录的欲望，每天在游戏里亲友群里 SNS 上说话的仿佛不是我，一旦坐在电脑前打开文档我就没办法把现实发生过的生活组织成通顺的文字记录下来，而在写出这句话之前我甚至都没有意识到这一点。
　　打个粗俗一点的比方，仿佛脑子上蒙了层猪油。
　　就连日常的对话能力也在退化，前两天打 FF 战场出来看见 hds 在群里晒战绩，然后我完全不过脑地来了一句“吐了，我刚喊有无 zc 都没人理我”，其实我本意只是想抱怨一下我俩好像进了同一个闸我家还被按在地上打得我很痛苦，直到 14 跳出来说他们只是在日随蹭经验，我才反应过来这句话怎么看怎么像我对她俩有意见了。
　　虽然快七夕了还在一个人孤寡战场确实很凄凉但我也是清楚人家 CP 日随跟我一点关系都没有的【
　　如果是用语言交流我大概至少能完整表达出我想说的意思，可惜打字交流不可知的因素太多了，多说一句解释又显得刻意，只能假装我什么都没想直接跳过这个话题……

　　写到这里脑子突然又宕机了。
　　说起来最近精神分裂的症状也有点抬头的迹象，不管做什么事都能听见有人在背后议论我，虽然现在我还能分清楚哪些声音是真的哪些是假的，但是再这样继续当山顶洞人下去会不会回到当年的状态也有点难说。
　　独居原来是，这么痛苦的一件事吗……_(:3」∠)_

　　也不是真的没有记录的欲望，大概还是我在逃避，不想承认明明每天起床都告诉自己要画画结果坐不到两个小时就冲到狼狱的雇员铃前面挂机【
　　虽然这几天也不是什么都没干，新搬的森 S 个人房差不多装好了，自我感觉还挺良好的。
　　然后……然后好像就没了。战场和打本几乎都是在打发时间，就这么打发着一天又一天，最近连歌都听不进去了。
　　也不知道我到底是在为了什么打发时间，大概只是单纯打发活着的时间吧。

　　也不是真的无欲无求，只是我想要的东西以我现在的高度都触及不到，可是倒回原来的样子又太过无趣，又找不到新的方向去努力些什么。
　　啊不行果然这样下去是不行的。我可能就是太惯着自己了才这么矫情兮兮的。明天开始把网断了去画画，不摸点什么东西出来不要再社交了。
　　每隔一段时间写点什么东西其实还有点像自我检修。可惜我大概是坏得太久已经记不清正常运行的时候是什么样的了。

　　突然又想到一件事，之前做过一个 canvas 的换装（头）小游戏模板，结果没放在文章归档序列里，在本地文件里找了一下，原来是新建了个一个不带链接的 page 藏起来了。
　　依稀记得当初是页面渲染有问题，实际把文章放进普通 post 文件夹里以后发现，我的 css 和 JS 都是单独写好了链进去的，如果不用 page 的格式渲染，这个 css 和 JS 文件都会被识别成 post 然后出现在文章流里。尝试了在文件名前面加下划线确实避免了这个问题，但整个文件都压根不处理了，自然也读取不出来 OTL
　　查了一下隐藏某篇文章需要魔改代码，我……算了，我现在人话都忘了怎么说，还看得懂代码吗？【？
　　折腾了一下把这个页面放到相册里去了，排版啥的以后再搞，至少我不会再忘了这东西在哪【
